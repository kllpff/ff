# Documentation

This directory contains complete documentation for FF Framework in two languages:

## Languages

- **[English Documentation](en/README.md)** - English version
- **[Документация на русском](ru/README.md)** - Russian version

## Structure

Each language directory contains:
- **Getting Started** - Installation and quick start guides
- **Core Concepts** - Routing, controllers, views, models
- **Database** - Database operations, migrations, seeding
- **Features** - Validation, security, sessions, caching
- **Advanced** - Middleware, dependency injection, configuration
- **Deployment** - Production deployment and troubleshooting

Choose your preferred language to get started!
